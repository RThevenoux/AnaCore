AnaCore - managing standard file formats and objects from NGS¶

Description¶

Anapath Core is a package containing libraries for managing standard file formats (BED, fasta, gff, gtf, MAF, newick, VCF, …) and objects from NGS.

Installation¶

pip install anacore
